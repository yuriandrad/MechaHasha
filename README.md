## Sobre o Projeto

**Mecha-Hasha** é uma ferramenta em Bash desenvolvida para:

* 🔍 Identificação de hashes
* ⚔️ Ataques de dicionário
* 🧪 Testes educacionais e auditorias autorizadas

## Funcionalidades

* Entrada de hash manual ou via arquivo
* Identificação automática com `hashid`
* Seleção de wordlists
* Suporte a:

  * `hashcat`
  * `john`
* Detecção automática de ferramenta
* Execução de ataque de dicionário
* Output de resultados

## Instalação

### Dependências

```bash
sudo apt update
sudo apt install hashcat john hashid
```


## Como usar

```bash
chmod +x mecha-hasha.sh
./mecha-hasha.sh
```
